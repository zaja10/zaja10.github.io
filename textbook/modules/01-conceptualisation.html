Understanding Genotype-by-Environment interactions requires separating absolute performance from relative stability.

  • (a) Performance Plot Emphasizes patterns of response across sites. The environment mean is used as a proxy index for absolute ecosystem productivity (Yates & Cochran / Finlay-Wilkinson).
  • (b) Discrimination Plot (Bar Charts) Provides a visual 'relative genotype performance index' (Ivory et al. 1991). Highlights performance trends and how environments elicit unique variance.
  • (c) Proximity Plot (Ordination) Spatial position displays performance similarity in strict Euclidean space. Closer distance = highly similar GxE profile. Discrepant distances = high rank instability.
